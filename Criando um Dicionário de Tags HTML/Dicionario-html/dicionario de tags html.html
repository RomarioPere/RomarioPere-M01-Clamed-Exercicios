Dicionário de Tags HTML

As tags h1, h2, h3, h4 - serve para para criar títulos e subtítulos.

A tag p - Serve para criar paragráfos.

A tag b - Serve para da quebra de linhas no texto.

A tag a - Serve para links.

A tag li - Serve para criar uma lista.

A tag ul - Serve para criar uma lista não ordenada.
